# Frontend

React storefront and admin dashboard for the e-commerce app, built with [Vite](https://vite.dev/), Redux Toolkit, and React Router.

## Scripts

| Command | Description |
| --- | --- |
| `npm run dev` | Start the Vite dev server (http://localhost:3001) |
| `npm run build` | Build for production into `dist/` |
| `npm run preview` | Preview the production build locally |

## Environment

Create a `.env` file in this folder:

```
VITE_API_URL=http://localhost:3000/api
```

If unset, the API defaults to `http://localhost:3000/api`.
